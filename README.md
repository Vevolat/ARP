# ARP欺骗攻击工具

## 免责声明

**重要提示：本工具仅供学习和研究目的使用。未经授权对他人网络设备进行攻击是违法行为。使用者需自行承担所有法律责任。**

本工具基于ARP欺骗原理实现，用于演示网络安全攻击技术。请仅在您拥有完全控制权的网络环境中使用此工具，切勿用于非法目的。

## 使用环境说明

### 系统要求
- Python 3.6或更高版本
- 管理员权限运行（Windows）或sudo权限运行（Linux）

### 依赖库
- scapy: 网络数据包处理库
- 其他标准Python库

### 安装依赖
```bash
pip install scapy
```

### 运行方式
```bash
# Windows下以管理员权限运行
python main.py

# Linux下使用sudo运行
sudo python main.py
```

## 功能特性

### 1. 多语言支持
- 中文界面
- 英文界面

### 2. 跨平台支持
- 支持Windows和Linux操作系统

### 3. 两种攻击模式
#### 模式1：扫描网段设备攻击
- 自动扫描局域网内在线设备
- 显示设备IP地址和MAC地址
- 选择目标设备进行ARP欺骗攻击

#### 模式2：直接输入IP地址攻击
- 直接输入目标IP地址或域名
- 自动测试目标连通性
- 防止误攻击本机或网关

### 4. 安全保护机制
- 防止攻击本机
- 防止攻击网关
- 支持设置攻击持续时间

### 5. 全局退出命令
- 在任何输入提示处输入"exit"可退出程序

## 使用说明

1. 运行程序后首先选择语言（中文/English）
2. 选择攻击模式（模式1或模式2）
3. 根据所选模式进行相应操作
4. 在任何输入提示处可输入"exit"退出程序

## 注意事项

1. 必须以管理员权限运行程序（Windows）或使用sudo运行（Linux）
2. 仅在授权的测试环境中使用
3. 攻击行为可能影响网络正常通信
4. 使用本工具造成的任何后果由使用者承担

## 技术原理

本工具基于ARP（Address Resolution Protocol）协议的漏洞实现欺骗攻击。通过向目标设备和网关发送虚假的ARP响应包，使得目标设备将攻击者的MAC地址误认为是网关的MAC地址，从而将数据包发送到攻击者处，实现中间人攻击。
